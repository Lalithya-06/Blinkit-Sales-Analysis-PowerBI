# Blinkit Sales Analysis Dashboard (Power BI)

## 📊 Project Overview
This project analyzes Blinkit sales data using Power BI to generate business insights and support decision-making.

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## 📈 Key KPIs
- Total Sales: $1.2M+
- Number of Items: 8500+
- Average Sales & Ratings

## 🔍 Key Insights
- Tier 3 cities contributed highest revenue
- Supermarket Type 1 generated maximum sales
- Medium-sized outlets performed best
